# mongotweet
Simple Tweet app using socket.io and mongodb

### Version
1.0.0

## Install Dependencies
```bash
npm install mongodb socket.io --save
```

## Connect MongoDB
```bash
mongod
```
and

```bash
mongo

show dbs

use mongochat
```

## Run Server
```bash
npm start  or  node server.js
```

## Run App
Open index.html
